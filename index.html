ABOUT ME



I live in Columbus Ohio. I am very fortunate to have grown up in an environment where I could get ahold of the newer technology coming out. Whether through working summer jobs or scavenging, I always wanted to decode the new systems and computers to see how they worked. At a young age, I was interested in how to talk to computers. Now, I can live that childhood fantasy!



Code that "just works" doesn't mean it's clean code. I strive to communicate with computers in the most efficient way possible, and that means I'm always learning. Technology is moving fast, and the only way to understand it is to always be learning something new.



Both my professional and personal growth are top priorities to me, and I am always striving to be better.

Muzify | Team Development
API • Node.JS • Express • MongoDB • JavaScript • HTML • CSS3

Muzify is a music site mirrored on Spotify. This app uses a disconnected API for the back end logic, with a front end in JavaScript to pull information and render as a Single Page Application.


A custom component library was created for this project to ensure homogenization of methods across all files. The app would render an embedded video of the song being looked for, which the user had complete control over.
